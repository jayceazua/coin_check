# Military Coin Check

Military Challenge Medallion Foundation. Please read [proposal](proposal.md).

 - Latest Node LTS Release (v10.16.0)
 - Node Version Manager (nvm)
 - Truffle v5.0.22
 - Solidity ^0.5.0
